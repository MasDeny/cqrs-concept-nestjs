### 기술 스택

**Frontend** <br/>

```js
    frontend = {
        programmer: sieukim,
        language: JavaScript,
        framework: React,
        library: [antd, Styled-Components],
    };
```

**Backend** <br/>

```js
    backend = {
        programmer: vvsos1,
        language: TypeScript,
        framework: NestJS,
        db: [MySQL, Redis],
        library: TypeORM, 
        searchEngine: [ElasticSearch, Logstash],
        fileStorage: AWS S3,
    };
    
    deployment = {
        NestJS: AWS EC2,
        MySQL: Azure Database for MySQL,
        ElasticSearch: Elastic Cloud,
        Logstash: Azure Virtual Machine,
        Redis: Amazon ElasticCache,
    }
```
